Image Alignment Template

Image Alignment boxes are a great way to incorporate visuals along side text. These boxes allow for easy text wrapping around the image.

To copy and paste:

To copy any of the following Image Alignment Boxes, make sure the page is in 'Edit' mode, select everything between start copy and end copy for the specific layout, then copy and paste into the HTML Editor on the new page.
