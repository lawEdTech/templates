Click Reveal

!! IMPORTANT -- animation details!!

Each Click Reveal component below has it's own ID in its HTML code. This allows for the Click Reveal buttons to work independently of each other. Due to this, if you want 2 Click Reveal buttons on the same page, you cannot copy one button and paste it twice. This would make both buttons expand and collapse at the same time. Instead, you must copy and paste 'Click Reveal 1', then copy and paste 'Click Reveal 2'. Please note that you can have a maximum of 6 Click Reveal buttons per page.

To copy and paste:

To copy any of the following Click 'n' Reveal buttons, make sure the page is in 'Edit' mode, select everything between start copy and end copy for the specific layout, then copy and paste into the HTML Editor on the new page.
